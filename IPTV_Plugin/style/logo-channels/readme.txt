Высота и ширина картинки для  не должна превышать 50x50px иначе искозится картинка
Так же плагин выкачивает картинку из файла m3u если она прописана и указан id канала

Пример:
#EXTINF:-1 tvg-logo="https://tvoetv.space/tv" tvg-id="tvoetv1",Твоё ТВ HD v1
http://tvoetv.space:8080/tvoetv/mpegts?token=MkP9_m4vd1

tvg-logo это прямая ссылка на логотип канала
tvg-id это id канала

При соблюдении такого синтаксиса будет скачиватся картинка и погружатся в список каналов.
